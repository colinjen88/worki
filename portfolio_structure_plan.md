## 主要頁面架構規劃

### 1. 首頁 (index.html)

#### 視覺動線規劃 (Z-Pattern + F-Pattern 結合)
```
[品牌標誌]                    [導航列]
         ↘
    [Hero 主視覺區 - 大標語 + 動態背景]
         ↓
[專業領域卡片列] ← → [滑動展示]
         ↓
[精選作品 - 瀑布流/網格] 
         ↓
[技能視覺化展示]
         ↓
[CTA 區塊 + 聯絡]
         ↓
[Footer]
```

**詳細區塊規劃：**

- **Hero Section** (首屏吸睛)
  - 全螢幕動態粒子/漸層背景 (Canvas/WebGL)
  - 大標語 Typography 動畫 (逐字浮現)
  - 副標語簡短描述專業定位
  - CTA 按鈕：「探索作品」+ 「聯絡合作」
  - 向下捲動提示動畫
  - **UX**: 3秒內傳達核心價值，避免過多文字

- **專業領域展示** (快速建立信任)
  - 4 欄卡片式設計 (UI/UX | Motion | Graphic | Dev)
  - 每張卡片：Icon + 標題 + 3個關鍵技能 Tag
  - Hover 效果：卡片浮起 + 微光掃過
  - 點擊後平滑捲動至對應作品區
  - **UX**: 一眼看出多元能力，降低認知負荷

- **精選作品展示** (視覺焦點)
  - 瀑布流 (Masonry) 或 不規則網格佈局
  - 每個作品：大圖預覽 + 專案名稱 + 類別標籤
  - Hover：圖片放大 + 資訊浮出 + "查看詳情" 提示
  - 無限捲動或 "載入更多" 按鈕
  - **UX**: 圖片優先，減少文字干擾，快速瀏覽

- **技能視覺化** (專業度證明)
  - 動態圓環進度條或長條圖
  - 工具圖標矩陣 (Figma, AE, VS Code 等)
  - 數字化成就展示 (X年經驗, Y個專案)
  - **UX**: 視覺化數據比文字更有說服力

- **聯絡 CTA** (轉換導向)
  - 對比色背景區塊突顯
  - 簡短有力的合作邀請文案
  - 直接 Email 連結 + 社群連結
  - **UX**: 減少步驟，一鍵聯絡

### 2. 作品集頁面 (portfolio/)

#### 視覺動線策略
```
[分類篩選列 - Sticky Header]
         ↓
[作品網格 - 依日期/熱門排序]
         ↓
[點擊進入 → 詳細頁]
```

**分類篩選系統 (導航清晰)**
- Sticky 頂部篩選列：All | UI/UX | Motion | Graphic | Web
- 滑動指示器 (Active 狀態底線動畫)
- 支援鍵盤快速切換 (1-5 數字鍵)
- URL hash 同步，可分享特定分類
- **UX**: 讓使用者快速找到感興趣的類別

**作品卡片設計 (視覺層級)**
- 統一比例：16:9 或 4:3 (保持畫面節奏)
- 圖片懶加載 + 模糊預覽占位
- 資訊層級：
  1. 作品圖 (最大視覺權重)
  2. 專案名稱 (粗體標題)
  3. 簡短描述 (1行，灰色次要)
  4. 技術標籤 (小字 pill 形狀)
- **動效**: 進入視窗時 stagger 淡入
- **UX**: 一致的卡片降低學習成本

**詳細頁面架構 (故事性動線)**
- **開場**: 大圖 Hero + 專案標題 + 角色標籤
- **專案資訊列**: 客戶 | 時間 | 角色 | 連結
- **背景挑戰**: 問題陳述 (文字 + 示意圖)
- **設計過程**: 流程圖/線框圖展示
- **最終成果**: 高品質圖片/影片展示
  - 圖片 Lightbox 燈箱查看
  - 影片自動播放 (靜音)
- **學習反思**: What I learned 區塊
- **上下篇導航**: 維持瀏覽動線
- **UX**: 講故事而非單純展示，建立情感連結

**互動效果規劃**
- 卡片 Hover: `transform: translateY(-8px) scale(1.02)` + 陰影加深
- 圖片 Hover: 輕微放大 `scale(1.05)` + 暗遮罩
- 頁面轉場: Shared Element Transition (相同元素銜接)
- 捲動觸發: 內容區塊依序淡入 (避免一次全顯示)

### 3. 關於頁面 (about/)

#### 視覺動線：個人品牌故事線
```
[大頭照 + 簡短自我介紹]
         ↓

         ↓
[技能視覺化區]
         ↓

         ↓
```

**設計建議**

- **技能展示**: 
  - 使用 Radar Chart (雷達圖) 展示多維度能力
  - 或圓形進度條群組
- **時間軸**: 垂直線連接，左右交錯卡片
- **UX**: 展現個性，不只是技能清單

## 全域視覺動線規範

### 導航系統 (Navigation)
- **固定頂部導航**: 捲動時背景 blur + 縮小高度
- **當前頁面標示**: 底線動畫或顏色變化
- **行動版**: 漢堡選單 + 全螢幕覆蓋導航
- **麵包屑**: 二級以上頁面顯示路徑

### 頁面轉場動畫
- **進入**: 內容由下方淡入 (Y: 30px → 0)
- **退出**: 快速淡出 (150ms)
- **作品詳細頁**: 圖片由卡片位置展開 (FLIP 動畫)

### 捲動行為
- **Lenis 平滑捲動**: 阻尼係數 0.1 (流暢不暈)
- **捲動進度條**: 頂部細線顯示閱讀進度
- **回到頂部**: 捲動 500px 後顯示浮動按鈕

### 響應式斷點
- **Desktop**: 1280px+ (完整效果)
- **Tablet**: 768px-1279px (簡化動效)
- **Mobile**: <768px (關閉複雜動效，確保效能)

#### 容器規範
- 最大寬度: 1280px (內容區)
- 內邊距: 16px (Mobile) / 24px (Tablet) / 32px (Desktop)
- 內容區塊間距: 80px-120px

### 動效系統

#### 動效哲學
- **目的性**: 每個動效都應該有功能意義
- **一致性**: 統一時間曲線和方向
- **克制**: 動效輔助內容，不搶戲

#### 時間曲線 (Easing)
| 名稱 | 曲線 | 用途 |
|------|------|------|
| ease-out | `cubic-bezier(0, 0, 0.2, 1)` | 進入動畫 |
| ease-in-out | `cubic-bezier(0.4, 0, 0.2, 1)` | 一般過渡 |
| spring | `cubic-bezier(0.34, 1.56, 0.64, 1)` | 彈性效果 |

#### 持續時間規範
| 類型 | 時間 | 用途 |
|------|------|------|
| micro | 150ms | Hover、Focus |
| short | 300ms | 狀態變化 |
| medium | 500ms | 頁面轉場 |
| long | 800ms | 捲動觸發動畫 |

#### 具體動效規範

**微互動 (Micro-interactions)**
- 按鈕 Hover: `scale(1.02)` + 陰影加深, 150ms
- 連結 Hover: 顏色過渡 + 底線滑入, 200ms
- 卡片 Hover: `translateY(-4px)` + 光暈效果, 200ms
- 圖片 Hover: `scale(1.03)` + 暗遮罩, 250ms

**頁面轉場**
- 內容進入: Y: 40px → 0, opacity: 0 → 1, 600ms, ease-out
- 內容退出: opacity: 1 → 0, 200ms
- Stagger 延遲: 每個元素間隔 100ms

**捲動觸發動畫**
- 觸發點: 元素進入視窗 20% 時
- 動畫類型: 淡入 + 上移 / 左滑入 / 右滑入
- 單次觸發: 不重複播放

**載入動畫 (Loading)**
- 骨架屏: 脈衝動畫 (shimmer effect)
- 品牌載入: Logo 動畫 2秒後進入
- 進度條: 漸層流動效果

## 圖片與媒體規範

### 圖片比例
- 作品展示: 16:9 (寬螢幕感)
- 專案截圖: 4:3 (自然比例)
- 個人照片: 1:1 (大頭照)

### 圖片處理
- 格式: WebP (優先), JPEG (備援)
- 載入: 模糊漸進式 (LQIP)
- 邊框: 無或 1px 細邊框
- Hover: 輕微放大 + 暗角

### 影片展示
- 自動播放: 是 (靜音)
- 循環: 是
- 控制項: Hover 時顯示
- 封面: 高品質靜態圖
