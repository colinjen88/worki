微互動動效套件

將分散的 hover/transition 行為收斂成一套可重用規範，讓動效「輔助理解」而不是「搶戲」。

Role: Motion Deliverable: Guidelines Focus: Easing / Timing

動效原則

  • 目的性：告訴使用者發生什麼事
  • 一致性：同類互動用同樣節奏
  • 克制：動效不遮蔽內容
回作品集

規範內容

包含 easing、duration、hover 行為、以及進場動畫的節奏建議，讓 UI 在不同頁面保持一致。

Easing

ease-out / ease-in-out / spring

Duration

150ms / 300ms / 500ms
